HOW TO IMPLEMENT FILTERING STUDENT LOGIC 
✅ Step 1: Create a "Bulk Name Match" Feature
React UI

Create a page:

Drive: [Select TCS Drive]

Paste Selected Student Names:
---------------------------------
Rahul Sharma
Priya Verma
Aman Gupta
---------------------------------

[Find Students]


Textarea input.

✅ Step 2: Backend Logic

Officer pastes text.

Backend:

Split by new line

Trim spaces

Search in database

Example Backend Logic (Node + MySQL)
router.post("/match-students", async (req, res) => {
  const { names } = req.body;

  const nameArray = names
    .split("\n")
    .map(name => name.trim())
    .filter(name => name.length > 0);

  const placeholders = nameArray.map(() => "?").join(",");

  const [students] = await db.execute(
    `SELECT * FROM Students WHERE name IN (${placeholders})`,
    nameArray
  );

  res.json(students);
});

⚠️ IMPORTANT PROBLEM: NAME MISMATCH

Sometimes:

Email says:

Rahul Kumar


Database has:

Rahul Kumar Sharma


Exact match fails.

✅ Better Solution (Use LIKE Matching)
SELECT * FROM Students
WHERE name LIKE '%Rahul%'


Or advanced:

SELECT * FROM Students
WHERE LOWER(name) LIKE LOWER(?)

🚀 BEST PRACTICE (Use Student ID Instead of Name)

If possible, instruct companies to send:

Student ID - Name


Example:

101 - Rahul Sharma
102 - Priya Verma


Matching by student_id is 100% safe.
